# 設計資料インデックス台帳

## 管理ルール

- 設計資料を新規作成する際は、このファイルにエントリを追加すること
- インデックス形式: `[A-Z][0001-9999]`（カテゴリ英字1文字 + 連番4桁）
- カテゴリ定義は `.claude/rules/file-indexing.md` を参照

## カテゴリ一覧

| プレフィックス | カテゴリ |
|---|---|
| A | Architecture（アーキテクチャ設計） |
| B | Backend（バックエンド設計） |
| C | Component（コンポーネント設計） |
| D | Data（データモデル設計） |
| E | External（外部連携設計） |
| F | Frontend（フロントエンド設計） |
| G | General（汎用設計資料） |
| S | Specification（機能仕様書） |
| T | Test（テスト設計） |
